## ğŸš€ Getting Started

### Prerequisites
- Microsoft Excel 2016+ (for .xlsx file)
- Microsoft Word 2016+ (for .docx file)
- Microsoft PowerPoint 2016+ (for .pptx file)
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/sales-performance-dashboard.git
cd sales-performance-dashboard
```

2. **Open the files**
- Excel: Open `sales_data_model.xlsx` and navigate to "KPI Dashboard" sheet
- Dashboard: Open `dashboard.html` in your web browser
- Documentation: Open `KPI_Definition_Document.docx`
- Presentation: Open `Executive_Summary.pptx`

### Usage

**Excel Data Model:**
- Navigate to "KPI Dashboard" for executive summary
- Explore "Sales Data" for transaction details
- Review "Products" and "Regions" for master data
- All metrics auto-update when source data changes

**Interactive Dashboard:**
- Open `dashboard.html` in any modern browser
- No installation or server required
- View real-time KPI cards and interactive charts
- Use filter buttons to explore different time periods

**PowerPoint Presentation:**
- Open for executive stakeholder meetings
- Each slide tells a data story
- Includes strategic recommendations
- Export to PDF if needed
