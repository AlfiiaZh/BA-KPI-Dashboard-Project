# Sales Performance Dashboard - Business Analyst Portfolio Project

## Project Overview
A comprehensive business intelligence solution designed for regional sales management, featuring interactive dashboards, KPI definitions, and executive reporting.

## Business Scenario
Regional sales managers needed:
- Real-time sales performance monitoring
- Product line analysis across territories
- Data-driven insights for strategic decision making
- Executive-ready reporting materials

## Deliverables

### 1. **KPI Definition Document** (`KPI_Definition_Document.docx`)
Professional documentation defining all key performance indicators:
- Total Revenue, Profit, and Margin metrics
- Average Order Value (AOV) and transaction volume
- Regional and product-level KPIs
- Calculation methodologies and business impact analysis
- Stakeholder roles and data sources

### 2. **Sales Data Model** (`sales_data_model.xlsx`)
Multi-sheet Excel workbook with:
- **KPI Dashboard**: Executive summary with automated calculations
- **Sales Data**: 4,312 transactions across 2024 (365 days of data)
- **Products**: Master data for 5 products across 3 categories
- **Regions**: Territory information for 4 sales regions
- Automated formulas linking all data sources
- Professional formatting with color-coded inputs/formulas

### 3. **Interactive Dashboard** (`dashboard.html`)
Web-based visualization dashboard featuring:
- 5 key KPI cards with year-over-year comparisons
- Regional revenue breakdown (bar chart)
- Product performance mix (doughnut chart)
- Monthly revenue trend analysis (line chart)
- Category margin comparison (stacked bar chart)
- Regional performance table with status indicators
- Responsive design with professional color scheme

### 4. **Executive Summary Presentation** (`Executive_Summary.pptx`)
6-slide PowerPoint deck including:
- Title slide with branding
- Key performance highlights with visual metrics
- Revenue distribution analysis by region
- Product line performance breakdown
- Strategic recommendations (4 action items)
- Next steps and contact information

## How to Use

### Excel Data Model
1. Open `sales_data_model.xlsx`
2. Navigate to "KPI Dashboard" sheet for executive summary
3. Explore "Sales Data", "Products", and "Regions" sheets
4. All metrics auto-update when source data changes

### Interactive Dashboard
1. Open `dashboard.html` in any web browser
2. View real-time KPI cards and charts
3. Use filter buttons to simulate time period selection
4. Hover over charts for detailed tooltips

### PowerPoint Presentation
1. Open `Executive_Summary.pptx`
2. Present to stakeholders or export to PDF
3. Customize slides as needed for specific audiences
